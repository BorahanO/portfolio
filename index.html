  My Work  
Space Omelette
2D Physics Puzzle Platformer
Previous Next
Web Play Github

Space Omelette is a 2D physics based puzzle platformer. This project is aimed to bring interest in space exploration to younger kids. Furthermore, this project has also taught us how to work in a team where I took the role of the concept & game designer.

The game is available for browser play by clicking the Web Play button above.

Engine: Unity
Platform: Browser
Team: 2 Developers & 3 Artists
Duration: 7 weeks

Minesweeper
Minesweeper made in javascript
Previous Next
Web Play Github

This is a version of Minesweeper made in Javascript. It's a game that I played many times before and thus wanted to try and make it myself. With this project I taught myself how to work with the floodfill algorithm and neighbors checking.

Current build is able to be played on browser via the Web Play button above.

Environment: Javascript
Platforms: Browser
Team: Solo
Duration: 4 Days
